# Iron Sight Mobile Betting Lab

Mobile-first betting companion for Iron Sight Stats.

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch and the **/(root)** folder, then save.
6. Wait a minute, then open the GitHub Pages URL on your iPhone.

## Update data

Replace `latest.json` with your newest export, commit the change, and refresh the site.

## Files

- `index.html` — the mobile betting tool
- `latest.json` — the current stats dataset
- `.nojekyll` — prevents GitHub Pages processing issues
