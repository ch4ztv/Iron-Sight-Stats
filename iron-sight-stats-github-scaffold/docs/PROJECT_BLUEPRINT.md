# Iron Sight Stats Public Project Blueprint

## Public repository purpose
This repository is the clean public-facing GitHub Pages project.

It should contain:
- read-only site code
- public-safe assets
- public-safe JSON data
- bracket pages
- documentation relevant to the public build

It should not contain:
- admin workflows
- parser tools
- API keys
- raw maintenance utilities
- local-only backups
- hidden dev panels from the private app

## Recommended Pages configuration
- Repository: `iron-sight-stats`
- Branch: `main`
- Folder: `/ (root)`
- Site URL pattern: `https://USERNAME.github.io/iron-sight-stats/`

## Public v1 sections
- Dashboard
- Standings
- Matches
- Players
- Teams
- Betting Lab
- Brackets
- Optional Matchup section if polished enough
