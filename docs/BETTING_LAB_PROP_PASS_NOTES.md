# Betting Lab Prop Pass

This update pivots the Betting Lab to a player-prop workflow:
- Team selector
- Player selector
- Split selector
- Line input
- Kills / K/D map+mode stat toggle
- Player picture + team logo header
- Market tracker with tabs for Maps 1-3, Series K/D, Map 1, Map 2, and Map 3
- Summary cards for samples, average, median, best, and line hit rate
- Recent hit/miss cards
- Map + mode lab with filters and log table
