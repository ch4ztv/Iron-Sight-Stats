# Step 2 — GitHub setup

## Recommended model
Use a dedicated public repository separate from the private master app.

## Repository settings
- Repo name: `iron-sight-stats`
- Visibility: Public
- Default branch: `main`
- GitHub Pages source: `main` / root

## Top-level files to keep at root
- `index.html`
- `404.html`
- `.nojekyll`
- `README.md`
- `.gitignore`
