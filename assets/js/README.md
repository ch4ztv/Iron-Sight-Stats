# JavaScript folder

Planned public app files:
- `app.js` — app bootstrap and initialization
- `data-loader.js` — fetch public JSON data
- `router.js` — section/tab routing state
- `ui.js` — shared UI helpers

Subfolders:
- `sections/` — render logic for Dashboard, Standings, Matches, Players, Teams, Betting, Brackets, Matchup
- `components/` — reusable UI modules such as cards, tables, filters, media helpers
