# Public image assets

Subfolders:
- `branding/` — logos, favicon sources, social preview images
- `logos/` — team logos
- `players/` — player portraits grouped by team slug
- `team-stats/` — static team stat images grouped by team slug
- `ui/` — arrows, icons, utility graphics

Naming rules:
- lowercase only
- hyphen-separated slugs
- avoid spaces
- prefer consistent extensions per asset category
