# Assets Folder

This folder contains the public app's static assets.

## Subfolders
- `css/` app stylesheets
- `js/` client-side scripts
- `img/branding/` logos, favicons, brand art
- `img/logos/` team logos
- `img/players/<team-id>/` player images by team
- `img/team-stats/<team-id>/` team stat graphics
- `img/ui/` generic UI icons/backgrounds
